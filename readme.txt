== Description ==

The DaySwitcher plugin let you choose a schedule for displaying your WordPress themes.

Simply configure the display theme schedule : choose the beginning and the ending hour of the theme to display, or the month. This way, you can have a theme with bright colors in the morning, and a theme with dark colors in the evening, for instance.


<a href="http://blog.thomascook.fr/demo/en/day-switcher/">Official DaySwitcher plugin page</a>.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `DaySwitcher` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
